JS-AJAX
========================

JS training
----------- 

* Сервлет принимает\отдает данные 'json'
* `паджинация` должна работать (даже через стрелки браузера без обновления окна)
* 5-10 записей на странице
* общее количество записей не большое ( например 30)
* статус записи read, unread
* при нажатии на сообщение popup с тенью
* сортировка сообщений по дате ( последнее первым)
* поиск подсвечивается часть слова
* попробовать использовать local Storage
* попробовать использовать прокси объект

учебный проект на [github](https://github.com/4ernookiy/js-ajax)
